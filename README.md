# 📌 Overview

The **Cinema Ticket Booking System** is a Java-based console application designed to streamline movie ticket booking operations.  
It allows users to browse movies, check seat availability, and book tickets efficiently using a connected MySQL database.

This project demonstrates:

- Core Java concepts
- JDBC connectivity
- Database operations
- Object-Oriented Programming
- Console-based user interaction
- CRUD functionality

---

# ✨ Features

## 🎟️ Ticket Booking
- Book movie tickets easily
- Real-time seat availability updates

## 🎬 Movie Management
- View available movies
- Show movie timings
- Manage movie records

## 👤 User Management
- Store and manage user booking information

## 🗄️ Database Integration
- MySQL database support using JDBC
- SQL scripts for schema & sample data

## ⚡ Console Interface
- Interactive menu-driven application
